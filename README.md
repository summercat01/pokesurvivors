# Pokemon Survivors

Vampire Survivors 스타일의 포켓몬 테마 로그라이트 생존 게임

---

## 기술 스택

| 항목 | 내용 |
|------|------|
| 렌더링 엔진 | Phaser 3 (v3.88.2) |
| 언어 | TypeScript |
| 번들러 | Vite |
| 화면 크기 | 390 × 844 (모바일 세로 고정) |
| 스케일 모드 | `Phaser.Scale.FIT` + `CENTER_BOTH` |
| 백엔드 | Supabase (로그인/세션) |

---

## 개발 환경

```bash
npm run dev     # 개발 서버 실행
npm run build   # 빌드 (tsc + vite)
```

개발자 모드 활성화: URL에 `/dev` 포함 또는 `?dev` 쿼리파라미터 추가

---

## 게임 시스템

### 화면 구성

- **상단 HUD** (70px): 타이머, 레벨, 웨이브, 킬 수
- **게임 영역** (642px): 플레이어 + 적 + 투사체
- **하단 HUD** (132px): HP바, 경험치바, 무기 슬롯 6칸, 장신구 슬롯 6칸

### 조작

| 입력 | 동작 |

| WASD / 방향키 | 이동 |

| ESC / P | 일시정지 |
| 자동 | 가장 가까운 적 공격 |

## 진화 시스템

몬스터볼 획득 시, 아래 조건이 충족되면 일반 강화 대신 **진화** 발생:

- **1차 진화**: 무기 Lv5 + 해당 타입 돌 Lv1 이상 보유
- **2차 진화**: 무기 Lv5 + 해당 타입 돌 Lv5 보유

진화 시 무기는 **Lv1로 초기화**되지만, 진화 Lv1이 이전 최종 Lv5보다 강합니다.

| 단계 | 데미지 배율 | 쿨다운 배율 | 기준 |
|------|------------|------------|------|
| 1차 진화 직후 Lv1 | 원본 ×3.5 | 원본 ×0.60 | 원본 Lv5(×3.0)보다 강함 |
| 2차 진화 직후 Lv1 | 원본 ×11.5 | 원본 ×0.36 | 1차 Lv5(×10.5)보다 강함 |

| 하이퍼볼 | 10분 보스 | 5개 | 5회 | +60G |

획득 시 게임 일시정지 → 강화 내역 패널 표시 → "터치하여 계속"으로 재개

강화 우선순위: **진화 조건 충족 시 일반 강화 대신 진화 발동**

## 영구 업그레이드

골드로 구매, `localStorage` 저장. 게임 시작 시 자동 적용.

| 업그레이드 | 효과 | Lv1 | Lv5 | 비용 (누적) |
|-----------|------|-----|-----|-------------|
| 공격력 | 전체 공격력 배율 | +10% | +50% | 2.5K~62.5K |
| 이동 속도 | 이동속도 | +10 | +55 | 2.5K~62.5K |
| 쿨타임 감소 | 쿨다운 단축 | -5% | -25% | 2.5K~62.5K |
| 최대 체력 | 최대 HP | +20 | +120 | 2.5K~62.5K |
| 방어력 | 받는 피해 감소 | +1 | +8 | 2.5K~62.5K |
| 체력 재생 | 초당 HP 회복 | +1/s | +7/s | 2.5K~62.5K |
| 경험치 획득 | 경험치 배율 | +10% | +60% | 2.5K~62.5K |
| 치명타 확률 | 치명타 확률 | +3% | +20% | 2.5K~62.5K |

총 최대 비용: **8종 × 62,500 = 500,000골드** (15분 게임당 ~5,000골드 기준 약 100판)
